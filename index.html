What is Counter.js?

Counter.js is a reusable UI component built with vanilla JavaScript OOP. It encapsulates all display logic, state management, and DOM manipulation into a single class so you never write the same counter logic twice.

01
⚙️

Encapsulated State

Each counter instance manages its own count independently. No global variables, no conflicts between instances.

02
🔒

Floor at Zero

The decrement method prevents the count from ever going below 0. Decrement and Reset buttons disable automatically.

03
🧱

DOM Separation

mount() builds the UI once. update() handles all re-renders. Clean separation of concerns on every instance.

04
🚀

Extendable

StepCounter extends Counter using extends and super — overriding only what it needs while reusing the rest.

What problem does it solve?

Without a reusable class, every counter on your page would require its own DOM queries, event listeners, and state logic written from scratch. Counter.js packages all of that into one clean, instantiable component you can drop anywhere.
