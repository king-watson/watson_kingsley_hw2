Check Out ME!

Counter.js is a reusable UI component built with vanilla JavaScript OOP. It encapsulates all display logic, state management, and DOM manipulation into a single class so you never write the same counter logic twice.

01
⚙️

Encapsulated State

Each counter instance manages its own count independently. No global variables, no conflicts between instances.

02
🔒

Floor at Zero

The decrement method prevents the count from ever going below 0. Decrement and Reset buttons disable automatically.

03
🧱

DOM Separation

mount() builds the UI once. update() handles all re-renders. Clean separation of concerns on every instance.

04
🚀

Extendable

StepCounter extends Counter using extends and super — overriding only what it needs while reusing the rest.

What Problem Does It Solve?

Without a reusable class, every counter on your page would require its own DOM queries, event listeners, and state logic written from scratch. Counter.js packages all of that into one clean, instantiable component you can drop anywhere.

Get It Up & Running In 3 Steps

Add HTML Containers

Create a div with a unique id where each counter will be rendered.

Import The Module

Use ES module imports in your JS entry file to pull in Counter and StepCounter.

Instantiate

Call new Counter() or new StepCounter() with your selector and starting values.

HTML 

<div id="counter-container1"></div>
<div id="counter-container2"></div>
<div id="counter-container3"></div>
<div id="counter-container4"></div>
JAVASCRIPT 
import { Counter, StepCounter } from './modules/counter.js';
                
const counter1 = new Counter('#counter-container1', 10);
const counter2 = new Counter('#counter-container2');
const counter3 = new StepCounter('#counter-container3', 0, 5);
const counter4 = new StepCounter('#counter-container4', 10, 3);

Counter( selector, initialValue )

selector CSS selector string for the container element. e.g. '#my-counter'
initialValue Starting count value. Defaults to 0 if not provided.

StepCounter( selector, initialValue, step )

selector CSS selector string for the container element.
initialValue Starting count value. Defaults to 0.
step Amount added or subtracted per click. Defaults to 1.
